== Description ==

Simplify WooCommerce order collaboration with our lightweight plugin! Leave and access quick notes directly in the order preview window, streamlining teamwork for shop managers and admins. Effortless order management at your fingertips.

Traditional order management in WooCommerce allows for notes, but the process can be cumbersome, requiring you to open each order individually. Our plugin transforms this experience by enabling admins to leave and view simple text notes directly within the order preview window. This means instant access to crucial information without the hassle of navigating through multiple screens.

Lightweight and user-friendly, our plugin is a must-have for teams managing orders on WooCommerce. It's a powerful tool that enhances teamwork by providing a quick and efficient way to communicate essential details about orders. Whether you're coordinating with multiple admins or streamlining your own workflow, our plugin ensures that you can leave and access important notes effortlessly.

No complex setup or unnecessary features — just a practical solution to optimize your order management process. 

### Features

* Quick Notes: Leave and access text notes instantly in the order preview window.
* Effortless Collaboration: Streamline teamwork among admins and managers for smoother order management.
* Lightweight Design: A simple and lightweight plugin that doesn't bog down your WooCommerce store.
* User-Friendly Interface: Intuitive design for easy adoption by store owners and team members.

== Installation ==

This plugin can be easily installed like any other WordPress integration by following the steps below:

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the orders page: WooCommerce > Orders
4. Click Order Preview icon and add note to the Order.

== Frequently Asked Questions ==

= How do I add a note? =

Navigate to WooCommerce Orders, click the Order Preview icon, and add or view notes for streamlined order management.

= Can I edit or remove notes? =

You can delete your own notes by clicking on the red 'x' icon in the upper corner of the note.

= Are notes visible to customers? =

No, these are private notes exclusively visible to order managers.

= Why don't I see notes when I open an order? =

At this early stage, notes are accessible only in the Preview Order modal window, not in the edit order screen. If you desire additional features or changes, please share your thoughts in the support forum. Your feedback matters!

= Can multiple admins leave notes simultaneously? =

Absolutely! Our plugin supports concurrent note additions by multiple admins, fostering real-time collaboration.

= Are there character limits for the notes? =

Currently, notes have a generous character limit to accommodate detailed information, promoting effective communication.

= What happens if I uninstall the plugin? Will I lose my notes? =

Uninstalling won't permanently delete your notes, as they are saved as order metadata. However, keep in mind that the notes won't be visible if the plugin is uninstalled. 

= Are there plans for additional features in future releases? =

Absolutely! We value user feedback. Visit our support forum to share your feature requests, and we'll consider them for upcoming releases.
